# Bibliography

Keep this manuscript repository self-contained by copying the needed verified
BibTeX entries from KnowledgeVault into `references.bib`.

Recommended source in KnowledgeVault:

```text
vault/citations/references.bib
```

When a citation comes from an absorbed vault paper, record its paper-note path,
`citation_key`, `bibtex_path`, and `citation_status` in
`../manuscript/citation-provenance.md`.
